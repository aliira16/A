Pré-remplir votre CV









1. Le CV en français, Anglais ou en Arabe
(السيرة الذاتية باللغة العربية او الانجليزية او الفرنسية )

2. L'emploi pour lequel vous postulez
(العمل الذي سوف تتقدم له )

3. Votre profil (Présentation de soi-même)
( (فقرن بسيط اذكر فيها منصبك مع قدرتك) خلفية عن نفسك )

4. Vos études et vos diplômes obtenus تعليمك ومؤهلاتك
(Veuillez joindre le(s) diplôme(s) détaillé(s) ainsi que les moyennes, الفترة الزمنية من - الى, نقاط ... ) )

5. Vos expériences professionnelles (travails, stages, bénévolat etc …)
(تجارب العمل مع التفاصيل الكاملة الفترةالزمنية والمكان و الخبرة الكاملة)

6. Vos compétences aquises, vos compétences sur lesquels vous travaillez encore.
( مهاراتك المكتسبة وحتى التي تعمل عليها )

7. Quelques informations requises par le type d'entreprise que vous souhaitez cibler
(بعض المعلومات التي يتطلبها نوع العمل الذي ترغب في استهدافه)

(اضف صورة شخصية)
